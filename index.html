Limitations and Interpretation

Accuracy

46.1 vs 54.2

Ours AAR is below R-CNN MAP, so this is not the strongest detector overall.

Action Space

Discrete moves are interpretable, but can be coarse for small or cluttered objects.

Reward

IoU is a clear signal, but it simplifies context, occlusion, and class ambiguity.

Best Reading

A conceptual proof that localization can be formulated as sequential decision-making.

Not: a direct replacement for modern object detectors.

But: strong evidence that a learned agent can search efficiently and visibly.

Main value: changing the question from scoring boxes to choosing actions.

Final Takeaway

Previous methods evaluated candidate regions.

This paper reformulates localization as policy-guided box transformation.

Strength

fewer evaluated regions, interpretable search

Limit

lower MAP than R-CNN, discrete actions
